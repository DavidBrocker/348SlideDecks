1. Mean Calculation

Sample Mean Calculation

A psychology researcher collects the following data on the number of hours five participants spent practicing a task: 3, 5, 8, 2, 7.

Question: What is the average number of hours spent practicing?

Manual Solution

\(\bar{x} = \frac{\sum(3+5+8+2+7)}{5} = \frac{25}{5} = 5\text{ hours}\)

Code Solution 
mean(c(3,5,8,2,7))
[1] 5

2. Mean Calculation

Sample Mean Calculation

A study measures the changes in mood scores before and after an intervention. The changes in mood scores are: -4, -2, 0, 3, 5.

Question: Calculate the mean change in mood score.

Manual Solution

\(\bar{x} = \frac{\sum(-4 + (-2) + 0 + 3 + 5)}{5} = \frac{2}{5} = 0.4\)

Code Solution 
mean(c(-4,-2,0,3,5))
[1] 0.4

3. Standard Deviation Calculation

Sample Standard Deviation

A psychologist measures the number of stressful events experienced by a small group of participants over a week: 2, 3, 6, 4, 5.

Question: Calculate the standard deviation of stressful events.

Manual Solution

  • First compute the sample mean: \(\bar{x} = \frac{2+3+6+4+5}{5} = \frac{20}{5} = 4.\)
  • Compute squared deviations: \((2-4)^2=4,\ (3-4)^2=1,\ (6-4)^2=4,\ (4-4)^2=0,\ (5-4)^2=1\)
  • Sum of squares = \(4+1+4+0+1 = 10\)
  • Sample variance \(s^2 = \frac{\sum (x_i-\bar{x})^2}{n-1} = \frac{10}{4} = 2.5)\)
  • Sample SD \(s = \sqrt{2.5} \approx 1.58\)

Code Solution 
sd(c(2,3,6,4,5))
[1] 1.581139

6. Z-Score Calculation

Z-Score for a Single Value

In a personality study, the average score on an extraversion scale is 50, with a standard deviation of 10. One participant scores 70.

Question: What is the Z-score for this participant?

Solution

\(z = \frac{X - \mu}{\sigma} = \frac{70 - 50}{10} = \frac{20}{10} = 2\)

Code Solution 
(70-50)/10
[1] 2

7. Z-Score Calculation:

Interpreting Z-Scores

A participant in a self-esteem survey has a Z-score of -1.5. The mean self-esteem score for the survey is 40, and the standard deviation is 5.

Question: Did this participant score above or below the mean, and by how many standard deviations?

Solution

A Z-score of -1.5 is below the mean. In raw score terms: \(X = \mu + \frac{z}{\sigma} = 40 + (-1.5)(5) = 40 - 7.5 = 32.5.\) So the participant scored 32.5, which is 1.5 standard deviations below the mean.

9. Z-Score Calculation

Multiple Z-Scores

In a clinical assessment, the average depression score is 60, with a standard deviation of 8. Two participants have scores of 72 and 48.

Question: Calculate the Z-scores for both participants and explain what the values tell us about their relative positions in the group.

Solution

  • For the first participant (72): \(z_1 = \frac{72 - 60}{8} = \frac{12}{8} = 1.5.\)
  • For the second participant (48): \(z_2 = \frac{48 - 60}{8} = \frac{-12}{8} = -1.5.\)
  • Interpretation: the first participant is 1.5 SD above the mean (higher depression score than average), while the second is 1.5 SD below the mean (lower depression score than average). They are symmetric around the mean.

Code Solution 
(72-60)/8;(48-60)/8
[1] 1.5
[1] -1.5

10. Normal Distribution & Areas

Probability Beyond a Given Z-Score

The distribution of scores on a math test is approximately normal with mean \(\mu = 100\) and standard deviation \(\sigma = 15\).

Question: What is the score of students who scored above \(z = 1.2\)?

Solution

\(z = \frac{x-\bar{x}}{s}\\ x = z\times\bar{x} + 100\\ 1.2\times15+100 = 118\)

Code Solution 
1.2*15 + 100
[1] 118
